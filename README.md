# NeuroPragma

## Installation
1. Install [REFramework for Pragmata with the C# scripting API](https://cursey.github.io/reframework-book/api_cs/general/index.html)
2. Install Newtonsoft.Json and copy the DLL to `Steam\steamapps\common\PRAGMATA\reframework\plugins\managed\dependencies\`
3. Copy the NeuroPragma DLL to `\Steam\steamapps\common\PRAGMATA\reframework\plugins\managed\`

Building the Project from source will automatically copy the DLLs to the correct location if you have Steam installed in the default location
